# Marketplace Simples

Este é um marketplace simples onde compradores podem descrever produtos que desejam comprar, e vendedores podem visualizar essas demandas.

## Funcionalidades

- **Tela Inicial**: Botões para escolher entre "Comprador" e "Vendedor".
- **Tela do Comprador**: Campo para descrever o produto desejado e enviar a demanda.
- **Tela do Vendedor**: Lista de todas as demandas de compra registradas.

## Como executar

1. Certifique-se de ter Python 3 instalado.
2. Execute o servidor: `python3 -m http.server 8000`
3. Abra o navegador e acesse `http://localhost:8000`

## Arquivos

- `index.html`: Tela inicial
- `buyer.html`: Tela do comprador
- `seller.html`: Tela do vendedor
